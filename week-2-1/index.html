Window Object

A window object is a global JavaScript object that represents the browser window that contains the web page. Browser Object Model (BOM) is a collection of properties and methods about the browser and computer screen. BOM enables us to query information about the browser being used, the screen size it is viewed in, previously visited pages, and many more. Several properties and methods are supported by all major browsers. These methods are available through the window object. Every browser window, tab, popup, frame, and iframe has a window object. Other javascript environments have a global object instead of a window object.

We do not need to use var, let, or const to declare global variables. These variables are properties of the global object (window object). The variables can do not need to refer to the window object when using them. ParseInt() and isNaN(), are global properties, so they are methods of a window object.

Dialogs

These functions produce dialogues in the browsers. They all pause the execution of a program and should be used wisely. Alert displays a message in a dialog box, confirm will display a confirmation dialog and a prompt request to a user for information.
