Forms

  • Forms are a common method of interacting with web pages. They are Google’s main component on their home page
  • They are made up of the form element.
  • Have controls like input fields, select menus, and buttons. The fields can be populated with data that can be processed once the form has been submitted.
  • The data can now be processed on the front end with javascript.
  • The form controls have values that are specified in the HTML, the user and javascript can change the values of the controls.
  • We can increase usability and accessibility by using correct and semantic mark-up and WAI -AIRA labels
