Git it Together

Command line git commands:

  1. git status – shows active git actions
  2. git pull – refreshes everything so your code is the same as the current repository
  3. git branch … – Creates a new branch with the given name
  4. git checkout … – Takes you to the specified branch
  5. git branch -a – shows all the active branches
  6. git branch -d … – deletes the specified branch
  7. git push origin –delete … – Deletes a remote branch
  8. git add . – Stages all the code for a commit. The period can be replaced with a desired file
  9. git reset head … – Undos the staging of the specified file.
  10. git commit -m “…” – Creates a new commit that can be merged onto the main branch. “-m” means message.
  11. git reset –soft HEAD~1 – Undos the last commit. You still need to unstage it too.
  12. git push origin master – Pushes the active commit to the main repository.
  13. git revert HEAD – Undos the last push by setting it as a new commit, still needs to be pushed again
  14. git diff – shows all differences between the repository and your current code
  15. git rebase is for personal work and not anything that is shared with others
  16. git clone … – Clones a repository using the link put at the end
  17. git fork … – Clones a repository and puts it on your own github account
